# TravelAI 前端项目

> TravelAI 智能旅游推荐与规划系统的前端模块

## 技术栈

- Vue 3 + Composition API
- TypeScript
- Vite
- Element Plus
- Vue Router
- Pinia
- Axios

## 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 生产构建
npm run build

# 预览生产构建结果
npm run preview
```

请参考项目根目录的 [README.md](../README.md) 获取完整的项目说明、配置指南和部署文档。

## 项目结构

```
src/
├── main.ts          # 应用入口
├── App.vue          # 根组件
├── router/          # 路由配置
├── stores/          # Pinia 状态管理
├── api/             # API 接口定义
└── views/           # 页面组件
```
